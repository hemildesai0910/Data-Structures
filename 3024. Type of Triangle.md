You are given an integer array `nums` of size **3**, where each element represents the **length of a side of a triangle**.

Your task is to determine the **type of triangle** that can be formed:

- **"equilateral"** â†’ All sides are equal.
- **"isosceles"** â†’ Exactly two sides are equal.
- **"scalene"** â†’ All sides are different.
- **"none"** â†’ The given sides **cannot** form a triangle.

> A valid triangle satisfies the **triangle inequality rule**:
> - `a + b > c`, `a + c > b`, and `b + c > a`.

**Explanation:**

## ğŸ’¡ Approach

1. First, **check triangle validity** using the **triangle inequality theorem**.
2. If not valid â†’ return `"none"`.
3. If valid:
   - All sides equal â†’ `"equilateral"`
   - All sides different â†’ `"scalene"`
   - Otherwise â†’ `"isosceles"`

class Solution {
public:
    string triangleType(vector<int>& nums) {
        bool valid = (nums[0] + nums[1] > nums[2]) &&
                     (nums[0] + nums[2] > nums[1]) &&
                     (nums[1] + nums[2] > nums[0]);

        if (!valid) {
            return "none";
        }

        if (nums[0] == nums[1] && nums[1] == nums[2]) {
            return "equilateral";
        } else if (nums[0] != nums[1] && nums[0] != nums[2] && nums[1] != nums[2]) {
            return "scalene";
        }

        return "isosceles";
    }
};
